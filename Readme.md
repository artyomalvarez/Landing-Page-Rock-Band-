# рҹҺё Oil Damage вҖ“ Landing Page

## рҹ“Ң Project Overview
This project is the result of a web development assessment focused on creating a **responsive Landing Page** for the rock band **Oil Damage**.

The main objective was to transform a base structure into a visually powerful website using semantic HTML5 and CSS3, following the **Mobile First** methodology.

## рҹҺҜ Objectives Achieved
- [x] **Semantic HTML5:** Proper use of tags such as `<header>`, `<nav>`, `<main>`, `<section>`, and `<table>`.
- [x] **Responsive Design:** Implementation of Media Queries to ensure an optimal experience on mobile and desktop (>1024px).
- [x] **Rock Aesthetic:** Visual identity based on dark tones and vibrant red highlights.
- [x] **Interactivity:** Fully functional hamburger menu for mobile devices.

## рҹҺЁ Style Guide
- **Colors:** Dark palette (`#0a0a0a`, `#1a1a1a`) with vibrant red accents (`#9f0712`, `#ff1f1f`) and neutral tones (`#ffffff`, `#a0a0a0`) to highlight the rock energy.
- **Typography:** Modern Sans-serif and bold weights for a clear visual hierarchy.
- **Layout:** Heavy use of **Flexbox** and **CSS Grid** for the header, Hero section, and content organization.

## рҹ“ұ Mobile First Design
Development was carried out by prioritizing mobile devices:
1. **Mobile:** Side/full-screen hidden menu and event table adapted for vertical reading.
2. **Desktop (>1024px):** Navigation menu expands horizontally and the table returns to its traditional data structure.

## рҹҡҖ How to Run the Project
1. Clone this repository or download the source files.
2. Ensure the folder structure shown above is maintained.
3. Open the `index.html` file in any modern web browser.

## рҹ“қ Developer Notes
A modular CSS architecture (split by sections) was chosen to facilitate maintenance and improve the scalability of the site.
